# Campaign Messages

Use campaigns for bulk, scheduled, or recurring sends with analytics.

## When to use

- Promotions, announcements, reminders to large lists.

## Workflow

1. Create a campaign with content, sender ID, and schedule.
2. Import or link recipient groups.
3. Launch and monitor delivery metrics.
