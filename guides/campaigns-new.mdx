# Campaign Messages

Campaigns provide a way to organize and manage bulk SMS messaging within workspaces. They help you group related messages and track your bulk messaging activities effectively.

## What are Campaigns?

Campaigns are containers within workspaces that help you organize your bulk messaging efforts. Each campaign belongs to a specific workspace and can be used to send messages to multiple recipients.

### Campaign vs Instant Messages

| Feature | Campaign Messages | Instant Messages |
|---------|------------------|------------------|
| **Organization** | Grouped under campaigns in workspaces | Individual messages |
| **Recipients** | Multiple phone numbers | Single phone number |
| **Management** | Created and tracked within campaigns | Sent directly |
| **Use Case** | Bulk communications, marketing | Notifications, alerts |

## When to Use Campaigns

### Bulk Communications
- **Marketing**: Promotional messages to customer lists
- **Announcements**: Company updates to multiple recipients  
- **Events**: Invitations and event notifications
- **Newsletters**: Regular updates to subscriber lists

### Organization Benefits
- **Project Tracking**: Group messages by initiative or project
- **Workspace Management**: Keep campaigns organized within workspaces
- **Bulk Sending**: Send the same message to multiple recipients efficiently

## Managing Campaigns

### Creating a Campaign

```bash
curl -X POST https://karibu.briq.tz/v1/campaigns/create/ \
-H "Content-Type: application/json" \
-H "X-API-Key: your_api_key_here" \
-d '{
  "workspace_id": "ws_abc123xyz789def",
  "name": "Summer Sale 2024",
  "description": "Promotional campaign for summer discounts and offers"
}'
```

**Response:**
```json
{
  "success": true,
  "campaign_id": "camp_xyz789abc123def",
  "workspace_id": "ws_abc123xyz789def",
  "name": "Summer Sale 2024", 
  "description": "Promotional campaign for summer discounts and offers",
  "created_at": "2024-08-10T10:30:00Z"
}
```

## Sending Campaign Messages

Once you have a campaign created, you can send messages to multiple recipients through it:

```bash
curl -X POST https://karibu.briq.tz/v1/messages/campaign/ \
-H "Content-Type: application/json" \
-H "X-API-Key: your_api_key_here" \
-d '{
  "campaign_id": "camp_xyz789abc123def",
  "phone_numbers": [
    "255712345678",
    "255787654321", 
    "255798765432"
  ],
  "message": "üèñÔ∏è Summer Sale: 30% off all items! Use code SUMMER30. Shop now: bit.ly/summer-sale",
  "sender_id": "YourStore"
}'
```

**Response:**
```json
{
  "success": true,
  "campaign_id": "camp_xyz789abc123def",
  "message_ids": ["msg_123", "msg_124", "msg_125"],
  "recipients_count": 3,
  "message": "üèñÔ∏è Summer Sale: 30% off all items! Use code SUMMER30. Shop now: bit.ly/summer-sale",
  "sender_id": "YourStore",
  "sent_at": "2024-08-10T10:30:00Z"
}
```

### Campaign Message Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `campaign_id` | string | Yes | ID of the campaign to send through |
| `phone_numbers` | array | Yes | List of recipient phone numbers |
| `message` | string | Yes | SMS message content |
| `sender_id` | string | Yes | Your brand name or sender identifier |

## Code Examples

### JavaScript/Node.js

```javascript
const axios = require('axios');

class BriqCampaigns {
  constructor(apiKey) {
    this.apiKey = apiKey;
    this.baseURL = 'https://karibu.briq.tz';
    this.headers = {
      'X-API-Key': apiKey,
      'Content-Type': 'application/json'
    };
  }

  async createCampaign(workspaceId, name, description = '') {
    try {
      const response = await axios.post(`${this.baseURL}/v1/campaigns/create/`, {
        workspace_id: workspaceId,
        name: name,
        description: description
      }, { headers: this.headers });
      
      return response.data;
    } catch (error) {
      console.error('Error creating campaign:', error.response?.data);
      throw error;
    }
  }

  async getAllCampaigns() {
    try {
      const response = await axios.get(`${this.baseURL}/v1/campaigns/all/`, {
        headers: { 'X-API-Key': this.apiKey }
      });
      
      return response.data;
    } catch (error) {
      console.error('Error fetching campaigns:', error.response?.data);
      throw error;
    }
  }

  async getCampaign(campaignId) {
    try {
      const response = await axios.get(`${this.baseURL}/v1/campaigns/${campaignId}`, {
        headers: { 'X-API-Key': this.apiKey }
      });
      
      return response.data;
    } catch (error) {
      console.error('Error fetching campaign:', error.response?.data);
      throw error;
    }
  }

  async updateCampaign(campaignId, updates) {
    try {
      const response = await axios.patch(`${this.baseURL}/v1/campaigns/update/${campaignId}`, updates, {
        headers: this.headers
      });
      
      return response.data;
    } catch (error) {
      console.error('Error updating campaign:', error.response?.data);
      throw error;
    }
  }

  async sendCampaignMessage(campaignId, phoneNumbers, message, senderId) {
    try {
      const response = await axios.post(`${this.baseURL}/v1/messages/campaign/`, {
        campaign_id: campaignId,
        phone_numbers: phoneNumbers,
        message: message,
        sender_id: senderId
      }, { headers: this.headers });
      
      return response.data;
    } catch (error) {
      console.error('Error sending campaign message:', error.response?.data);
      throw error;
    }
  }
}
```

### Python

```python
import requests

class BriqCampaigns:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = 'https://karibu.briq.tz'
        self.headers = {
            'X-API-Key': api_key,
            'Content-Type': 'application/json'
        }
    
    def create_campaign(self, workspace_id, name, description=''):
        """Create a new campaign"""
        data = {
            'workspace_id': workspace_id,
            'name': name,
            'description': description
        }
        
        response = requests.post(
            f'{self.base_url}/v1/campaigns/create/',
            json=data,
            headers=self.headers
        )
        
        return response.json()
    
    def get_all_campaigns(self):
        """Get all campaigns"""
        response = requests.get(
            f'{self.base_url}/v1/campaigns/all/',
            headers={'X-API-Key': self.api_key}
        )
        
        return response.json()
    
    def get_campaign(self, campaign_id):
        """Get specific campaign details"""
        response = requests.get(
            f'{self.base_url}/v1/campaigns/{campaign_id}',
            headers={'X-API-Key': self.api_key}
        )
        
        return response.json()
    
    def update_campaign(self, campaign_id, updates):
        """Update campaign details"""
        response = requests.patch(
            f'{self.base_url}/v1/campaigns/update/{campaign_id}',
            json=updates,
            headers=self.headers
        )
        
        return response.json()
    
    def send_campaign_message(self, campaign_id, phone_numbers, message, sender_id):
        """Send message through campaign"""
        data = {
            'campaign_id': campaign_id,
            'phone_numbers': phone_numbers,
            'message': message,
            'sender_id': sender_id
        }
        
        response = requests.post(
            f'{self.base_url}/v1/messages/campaign/',
            json=data,
            headers=self.headers
        )
        
        return response.json()

# Usage example
campaigns = BriqCampaigns('your_api_key_here')

# Create a holiday campaign
campaign = campaigns.create_campaign(
    'ws_abc123xyz789def',
    'Holiday Greetings 2024',
    'End of year holiday messages to customers'
)

print(f"Campaign created: {campaign['campaign_id']}")

# Prepare recipient list
recipients = [
    '255712345678',
    '255787654321',
    '255798765432'
]

# Send holiday message
message = "üéÑ Happy Holidays from our team! Thank you for being a valued customer this year."

    campaign['campaign_id'],

    'YourBrand'

print(f"Holiday message sent to {result['recipients_count']} customers")
print(f"Message IDs: {result['message_ids']}")
```

## Best Practices

### Campaign Organization
1. **Descriptive Names**: Use clear, specific campaign names
2. **Workspace Structure**: Group related campaigns in appropriate workspaces
3. **Documentation**: Add meaningful descriptions to campaigns
4. **Regular Cleanup**: Archive old campaigns to keep lists manageable

### Message Content
1. **Clear Purpose**: Make the campaign intent obvious in the message
2. **Brand Consistency**: Use consistent sender IDs across campaigns
3. **Call to Action**: Include clear next steps for recipients
4. **Length Management**: Keep messages concise but informative

### Recipient Management
1. **Phone Number Format**: Ensure proper formatting with country codes
2. **List Validation**: Verify phone numbers before sending
3. **Batch Size**: Consider sending limits and split large lists if needed
4. **Opt-out Compliance**: Respect recipient preferences and regulations
