# Webhooks & Callbacks

Receive asynchronous events such as message delivery updates.

## Configure

- Set a webhook URL at the workspace level.
- Verify with a shared secret or signature when available.

## Retry

BRIQ retries failed deliveries with exponential backoff.

## Security

- Use HTTPS and validate signatures when provided.
