# Workspaces

Workspaces organize projects, credentials, and access control.

## Create a workspace

Use the dashboard or the API to create and configure a workspace. Assign a name, default sender ID, and optional webhook URL.

## Developer access

- Grant developer access to limit API scope to development environments.
- Rotate keys and remove users when offboarding.
