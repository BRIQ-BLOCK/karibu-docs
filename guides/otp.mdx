# OTP

Use the OTP APIs for authentication flows such as login, signup, or sensitive actions.

## Request

Trigger an OTP to a phone number with expiry and channel settings.

## Verify

Validate the code and consume it; lock after repeated failures.

## Resend

Apply rate limits and cooldowns to prevent abuse.
