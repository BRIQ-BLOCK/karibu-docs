## Create Your BRIQ Account

Get started with the BRIQ API for messaging, verification, and authentication via SMS, Swahili voice, and WhatsApp. Follow these steps to register and access your API credentials.

<Card
  title="Sign Up Now"
  icon="user-plus"
  href="https://briq.tz/register"
  horizontal
>
  Create a free BRIQ account in minutes to access our multi-channel API.
</Card>
![Alt text describing the image](/images/signup.png)

## Request SenderID

Once you have created your account, you can request a SenderID from the BRIQ dashboard.

![Alt text describing the image](/images/senderID.png)

## Generate API Key

Once you have created your account, you can generate an API key from the BRIQ dashboard.

![Alt text describing the image](/images/apikeys.png)

<Note>
  We advice you to use the API key for production and the test API key for testing.
</Note>
