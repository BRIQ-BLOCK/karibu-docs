BRIQ provides a simple and secure way to send SMS messages.

### Features

- **Messaging**: send SMS, WhatsApp, and voice messages.
- **Personalization**: customize messages with dynamic fields.
- **Verification**: send and verify one-time passcodes (OTPs).

### Endpoints

- **Send SMS**: `/v1/message/send-instant`
